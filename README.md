# Ordak web browser

This is a personal project of mine, which is supposed to become a web browser for Android OS.
<br/>

The initial project was forked from [DuckDuckGo](https://github.com/duckduckgo/Android) android app,
but has diverged from that significantly.

## Naming Convention

`Ordak` is Persian equivalent of __duck__ and since it was first forked from DuckDuckGo, I thought
it's going to be a good name for this project.
